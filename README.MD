## NodeJS Tutorial

### Setup:

```console
git clone https://github.com/thaiphuoc1303/nodejs_tutorial.git
npm init
npm i express
npm install nodemon --save-dev
npm i morgan --save-dev
npm install express-handlebars
npm i node-sass --save-dev

npm start //start server
npm run watch
```
